# WorldWise 🌍

WorldWise is a travel tracking application that allows users to mark and save destinations they have visited on an interactive map. Users can click on any location, add information about the city and country, and keep a visual record of their travels around the world.

## Features

* 🗺️ Interactive world map
* 📍 Add visited destinations by clicking on the map
* 🏙️ Store city information
* 🌎 Track countries you have visited
* 📋 View a list of saved cities and countries
* ⚡ Fast and responsive user interface built with React

## Tech Stack

* React
* React Router
* Context API
* CSS Modules
* Leaflet Maps
* JSON Server (Mock Backend)
* Vite

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Sagarsj19/worldwise.git
```

2. Navigate to the project directory:

```bash
cd worldwise
```

3. Install dependencies:

```bash
npm install
```

4. Start the JSON Server:

```bash
npm run server
```

5. Start the development server:

```bash
npm run dev
```

## Usage

1. Open the application in your browser.
2. Click anywhere on the map to add a new destination.
3. Enter details about the city you visited.
4. Save the location and view it in your travel history.
